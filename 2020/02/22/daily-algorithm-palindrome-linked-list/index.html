Palindrome

palindrome이란 순서과 정방향과 역방향 모두 동일한 것을 의미한다.
간단한 예제를 들어보자면, 올해 2월인 ‘2020-02-02’날도 순서를 거꾸로 하면 ‘2020-02-02’로 동일해진다.

Problem

Linked-list에 대하여 palindrome이 성립하는 지 확인하라. 문제 출처
단 시간 복잡도는 O(n), 공간 복잡도는 O(1)이 되도록 해보자.
ex)
Input: 1 -> 2 (Linked-List)
Output: false

Input: 1 -> 2 -> 2 -> 1
Output: true

Solving

가장 기본적인 방법으로 재귀를 통해 문제를 해결하려 했다.
재귀의 누적에 따라 배열을 매개변수로 넘겨줘서 계산하도록 하는 것이다.

하지만 특정 케이스에 대해 시간 초과가 되어 문제를 해결할 수 없었다.

공간 복잡도를 줄이기 위해서는 게속해서 값을 덮어씌워 공간을 줄이는 것이 답이므로 해당 사항을 적용하고, 절반을 나눠 앞 뒤를 비교하는 것으로 시간 복잡도를 완화하면 다음과 같다.

소감

표기상 easy로 되어있어 단기간에 풀수 있을 것으로 믿었지만, 공간/시간 복잡도를 고려하고 해당 사항을 제한한 테스트 케이스를 통과 못하는 경우를 보면서, 이러한 요구사항을 고려하면서 문제를 푸는 실력을 기르는 것이 중요하다는 것을 느낄 수 있었다.
