medium

중위값(medium)이라는 것은 통계에서 해당 데이터의 분포 중 정 가운데의 값을 의미한다.
흔히, 평균값이 해당 데이터 분포의 중앙값으로 판단하는 경향이 있지만 실제로 양 극단의 분포가 차이가 나는 경우 이가 맞지 않는 경우가 발생한다.
값의 분포를 극단적으로 가정하여 평균값과(average) 중위값(median)값의 개념 차이를 비교해보자.
9명의 학생이 각각 100원짜리 동전을, 한명의 학생이 500원짜리 동전을 가졌다고 가정해보자.([100, 100, 100, 100, 100, 100, 100, 100, 100, 500])
이들의 평균은 (100 * 9 + 500 * 1) / 10 = 140로 140원이지만 실제로 평균보다 적은 동전을 가진 학생이 9명이 되는 것이다.
하지만 중위값으로 보면 이러한 데이터 분포의 정중앙값을 가르키므로 (100 + 100) / 2 =100원이 되고 이는 평균값보다는 진정한 의미의 중간값에 더 가깝다고 볼 수 있다.

Problem

Data Stream 내에서 숫자를 더하는 메서드(addNum)와 중위값을 구하는 메서드(findMedian)를 구현하여라. (문제 출처)

Solving

우선 인스턴스를 선언할 때는 빈 배열을 생성하여 객체를 초기화한다.
그리고 addNum에서 배열에 값을 넣고, findMedian에서 데이터를 정렬하고 중위값을 구하고자 하였다.

이 문제풀이에서는 addNum 메서드에서 sorting을 진행하였는데, binary search의 방식으로 해당 데이터의 저장 위치를 파악하여 데이터를 저장하였다.
